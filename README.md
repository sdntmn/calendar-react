# Создание проекта

- npx create-react-app calendar-react
- npm install --global gulp-cli
- npm i gulp
- npm i gulp-autoprefixer

- npm i gulp-sourcemaps
- npm i gulp-nodemon
- npm i gulp-livereload
- npm i gulp-stylus
- npm i css-loader
